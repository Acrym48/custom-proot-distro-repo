# Репозиторий пользовательских конфигураций Proot Distro

Коллекция конфигураций Proot Distro для различных Linux-дистрибутивов. Этот репозиторий предоставляет готовые к установке конфигурации, которые можно использовать с командой `proot-distro` в Termux.

## Что такое Proot Distro?

Proot Distro — это утилита для Termux, которая упрощает установку и управление различными Linux-дистрибутивами в chroot-окружении. Она позволяет запускать собственные дистрибутивы GNU/Linux без необходимости в root-правах на вашем устройстве.

## Установка

Чтобы установить конфигурации из этого репозитория:

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/My-proot-distro-repo.git
cd My-proot-distro-repo

# Запустить скрипт установки
./install.sh
```

Скрипт установки скопирует все файлы конфигурации из каталога `distros` в `$PREFIX/etc/proot-distro/`.

## Использование

После установки вы можете использовать установленные дистрибутивы с помощью:

```bash
# Вывести список доступных дистрибутивов
proot-distro list

# Установить дистрибутив (пример)
proot-distro install <имя-дистрибутива>

# Войти в дистрибутив (пример)
proot-distro login <имя-дистрибутива>
```

## Сотрудничество
